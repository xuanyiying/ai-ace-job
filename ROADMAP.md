# Roadmap

This document outlines the development roadmap for the AI Resume Optimizer project.

## Current Status (v1.0.0)

- [x] **Core Resume Analysis**: AI-powered parsing and analysis
- [x] **Resume Optimization**: Content improvement suggestions
- [x] **PDF Generation**: Professional resume templates
- [x] **Interview Prep**: AI mock interviews and question prediction
- [x] **Multi-Provider Support**: OpenAI, Qwen, DeepSeek, Gemini, Ollama
- [x] **Payment System**: Stripe and Paddle integration

## Short-term Goals (Q1 2026)

- [ ] **Mobile App Beta**: Release React Native mobile app for iOS and Android
- [ ] **Browser Extension**: Chrome extension for importing LinkedIn profiles
- [ ] **More Templates**: Add 10+ new resume templates
- [ ] **Community Features**: User profile sharing and community feedback
- [ ] **Localization**: Support for more languages (Spanish, French, German)

## Mid-term Goals (Q2-Q3 2026)

- [ ] **Job Matching Engine**: AI-powered job recommendations based on resume
- [ ] **Auto-Apply**: Automated job application submission
- [ ] **Enterprise Dashboard**: Team management and analytics for recruiters
- [ ] **API Access**: Public API for third-party integrations
- [ ] **Plugin System**: Allow community to create plugins

## Long-term Vision

- **Career Coach AI**: A comprehensive AI career companion that guides users through their entire career journey, from education to retirement.
- **Global Talent Marketplace**: Decentralized platform connecting talent with opportunities directly.

## Feature Requests

We welcome feature requests from the community! Please submit them via [GitHub Issues](https://github.com/xuanyiying/ai-resume/issues) with the "feature request" label.
