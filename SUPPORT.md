## Community Support

### GitHub Issues

For bug reports and feature requests, please use [GitHub Issues](https://github.com/xuanyiying/ai-resume/issues).
- Search existing issues first to see if your problem has already been reported.
- Use the appropriate issue template.
- Provide as much detail as possible.

### Community Channels

- **Discord**: Join our [Discord Server](https://discord.gg/ai-resume) to chat with other users and developers.
- **Discussions**: Use [GitHub Discussions](https://github.com/xuanyiying/ai-resume/discussions) for general questions and ideas.
